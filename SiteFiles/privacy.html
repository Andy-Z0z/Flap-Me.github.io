Политика конфиденциальности

1. Сбор и обработка данных

  • Сайт собирает только минимальные данные: имя и адрес электронной почты при регистрации для доступа к бета-материалам.
  • Данные используются исключительно для идентификации пользователя и предоставления доступа к разделу инсайдеров.

2. Хранение и безопасность

  • Данные хранятся локально в браузере пользователя и не передаются третьим лицам.
  • Владелец сайта не несёт ответственности за утечку данных, произошедших по вине пользователя.

3. Cookies и трекинг

  • Сайт не использует cookies для отслеживания активности пользователей.
  • Статистика посещений ведётся только в обезличенном виде (например, через GitHub Pages).

4. Удаление данных

  • Пользователь может удалить данные самостоятельно, выйдя из инсайдерского раздела (кнопка "Выйти из бета-доступа").

5. Связь
